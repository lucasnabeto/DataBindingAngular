Interpolação e Property Binding

String renderizada com interpolação: {{ url }}

O resultado de 1 + 1 é {{ 1 + 1 }}.

O resultado de 1 + 1 é {{ 1 + 1 + getValor() }}

  • Passa o valor do componente para o template.
  • Usa-se colchetes para fazer o binding (ex.: <img [src]="urlImagem">)
  • Formato padrão é através de bind-nomePropriedade (ex.: <img bind-src="urlImagem">)
  • Quando não existe uma propriedade no elemento, usa-se [attr.nomePropriedade] (ex.: [attr.colspan]).
